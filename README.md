A simple snake game written in Google's DART language for the hackathon @ Thoughtworks.

### Supported browsers (Tested browsers):
* Safari 5
* Firefox 12
* Chrome 18

* IE 6 and 7 are not supported.

### Hosted version: 
http://snake-dart-blr-hack.appspot.com/
